# Анализ генов, связанных с альбинизмом

## 1. Выбранный фенотип

В работе рассматривается фенотип - oculocutaneous albinism (альбинизм).

## 2. Выбранные гены

Для анализа выбраны два гена человека, связанные с данным фенотипом:
- OCA2
- TYR

Для каждого гена были получены mRNA последовательности человека и мыши из базы данных NCBI.

## 3. Методика

Для сравнения последовательностей использовались инструменты:
- Needle
- Water

Выравнивание выполнялось для каждой пары:
- человек - мышь (OCA2)
- человек - мышь (TYR)

## 4. Оценка выравниваний

Качество выравниваний оценивалось по проценту совпадений (Identity), количеству пропусков (Gaps) и общей длине выравнивания.

По результатам локального выравнивания для гена OCA2 сходство между последовательностями человека и мыши составило около 74%, для гена TYR - около 76%. Количество пропусков в этих выравниваниях находится на уровне примерно 9%, что можно считать умеренным значением.

Для гена OCA2 результаты глобального и локального выравниваний существенно различаются. Глобальное выравнивание показывает низкий процент совпадений и большое количество пропусков, что указывает на слабое выравнивание по всей длине последовательности. В то же время локальное выравнивание дает высокий процент совпадений и небольшое число пропусков, так как учитывает только наиболее похожие участки.

Для гена TYR наблюдается аналогичная ситуация: локальное выравнивание более точно отражает сходство последовательностей, тогда как глобальное выравнивание ухудшается из-за большого числа несовпадающих участков.

Локальное выравнивание оказалось более информативным, так как позволило выявить наиболее сходные участки последовательностей. Глобальное выравнивание в этих случаях искажается из-за различий в длине и структуре последовательностей.
